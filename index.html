サイトのコミットスクリプト

このサイトをコミットするためのシェルスクリプトを作った。これでコミットが楽になる。

GitHubへのpush

GitHubにpushする時に毎回ユーザー名とパスワードを求められるのが面倒なので、sshを使おうと鍵を生成したり何なりしたのだが、どうやらhttpsでもパスワードを保存出来たらしい。下のコマンドでパスワードが保存されるようになる。

$ git config --global credential.helper osxkeychain
